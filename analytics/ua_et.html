UA Event Tracking

Analytics event (either UA or GA4) should not be considered exclusively as the way to define a clicked element; however in most cases it is so. With event tracking you can trace any action that happened on the web-page, so let's take a look at few examples.

Definition


Onclick event occurs when the user clicks on an element.

Onsubmit event occurs when a form is submitted.

Onload event occurs when an object has been loaded.

Onplay and onpause events occur when the audio/video has been started/paused accordingly

Action






When this page loaded, it triggered an event
